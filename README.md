# 202601_Sato_Clinic
　〈案件概要〉

佐藤医院様よりサイトリニューアルをご依頼いただき、TOPページのデザイン・構成を制作しました。
依頼書と仕様書をもとに、ユーザーが必要な情報へすぐにたどり着ける導線設計と、医院らしい清潔感・安心感が伝わるデザインを意識しています。


　〈この案件で意識したこと〉

診療時間・電話予約・お知らせ・アクセスなど、来院前のユーザーが必要とする情報を優先的に整理し、初めて訪れる方でも迷わず確認できる構成を目指しました。
また、白を基調にした配色と余白設計によって、医療機関にふさわしい信頼感と親しみやすさの両立を意識しました。


　〈制作ポイント〉

1.情報の優先順位整理
　診療時間・電話予約・お知らせなど、確認頻度の高い情報を上部に配置し、視認性を高めました。

2.医院らしいトーンのデザイン設計
　白を基調とし、清潔感と安心感が伝わるように設計しました。

3.読みやすいTOPページ構成
　文章量、余白、セクションのまとまりを調整し、初見でも内容を把握しやすいページにしました。
